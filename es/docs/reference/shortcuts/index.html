Atajos de Teclado

Referencia completa de atajos de teclado en Rayforge.

Convenciones de Plataforma
  • Linux/Windows: Tecla Ctrl
  • macOS: Tecla ⌘ (Comando)
  • La documentación usa Ctrl - usuarios de macOS sustituyan con ⌘

Personalizando Atajos​

Atajos Personalizados

Los atajos de teclado están actualmente codificados. La configuración de atajos personalizados puede añadirse en futuras versiones.

Limitaciones actuales:

  • Los atajos no se pueden cambiar sin modificar el código fuente
  • No hay GUI para personalización de atajos
  • Algunas acciones pueden no tener atajos asignados

Solicitud de función: Si necesitas atajos personalizados, por favor abre un issue en GitHub.

Consejos y Trucos​

Consejos de Eficiencia​

  1. Aprende primero los atajos de edición - Ctrl+Z/Y/C/V/D se usan constantemente
  2. Usa alternadores de una sola tecla - H, T, P para cambios rápidos de vista
  3. Atajos de vista 3D - 1, 2, 7 para cambio instantáneo de vista
  4. Atajos de alineación - Shift+Flecha más rápido que hacer clic en botones de alineación
  5. Modo simulación - Ctrl+Shift+S para verificar rápidamente la ejecución

Atajos de Flujo de Trabajo​

Ciclo de edición rápida:

1. Nuevo Bosquejo (Ctrl+N) o Abrir (Ctrl+O)
2. Organizar elementos (teclas de flecha, Shift+Flecha para alineación)
3. Duplicar partes (Ctrl+D)
4. Verificar en 3D (F12)
5. Simular (F11)
6. Guardar (Ctrl+S) o Exportar (Ctrl+E)

Posicionamiento de precisión:

1. Seleccionar elemento
2. Teclas de flecha para empujones de 1mm
3. Shift+Flecha para saltos de 10mm
4. Shift+Inicio/Fin para centrar

Atajos Ocultos​

Algunos atajos menos conocidos:

  • Ctrl+, - Acceso rápido a preferencias (estándar en macOS, también funciona aquí)
  • Ctrl+Shift+Z - Rehacer alternativo (para usuarios que prefieren esto sobre Ctrl+Y)
  • Alt+A - Ajuste píxel perfecto para colocación precisa
  • Alt+T - Colocación rápida de pestañas equidistantes

Diferencias de Plataforma​

Linux​

  • Usa el modificador Ctrl estándar
  • Todos los atajos funcionan como documentado
  • Los atajos estándar GTK aplican en campos de texto

Windows​

  • Usa el modificador Ctrl estándar
  • Idéntico a los atajos de Linux
  • Las teclas específicas de Windows (tecla Win) no se usan

macOS​

  • Ctrl se mapea a ⌘ (Comando)
  • Alt se mapea a ⌥ (Opción)
  • Aplican las convenciones estándar de macOS
  • Cmd+Q para salir (en lugar de Ctrl+Q)

Solución de Problemas de Atajos​

Atajo No Funciona​

Problemas comunes:

  1. Foco en elemento equivocado - Asegúrate de que el lienzo o ventana principal tenga foco, no un campo de texto
  2. Aplicación en conflicto - Otra app puede estar interceptando el atajo
  3. Atajo de entorno de escritorio - Los atajos del sistema pueden anular (ej., Alt+F4)
  4. Teclado numérico vs fila numérica - Usa la fila numérica para atajos de vista 3D, no el teclado numérico

Soluciones:

  • Haz clic en el lienzo para asegurar que tenga foco
  • Revisa los atajos de teclado del sistema para conflictos
  • Prueba la acción del menú en su lugar para verificar funcionalidad

Teclas Modificadoras​

Si Ctrl no funciona:

  • Verifica que Bloq Mayús esté desactivado (puede interferir en algunos sistemas)
  • Prueba Ctrl en ambos lados del teclado
  • Revisa los ajustes de distribución del teclado

Si Alt no funciona:

  • Algunos gestores de ventanas capturan Alt para arrastre de ventanas
  • Prueba deshabilitando los atajos Alt del gestor de ventanas
  • Usa elementos del menú como alternativa
