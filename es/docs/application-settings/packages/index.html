Paquetes

La página de Paquetes muestra los paquetes de extensión instalados y te permite gestionarlos.

Resumen​

Los paquetes son extensiones que añaden funcionalidad a Rayforge. Pueden proporcionar:

  • Nuevas operaciones
  • Filtros de importación/exportación
  • Controladores de máquina
  • Mejoras de interfaz de usuario
  • Bibliotecas de materiales

Paquetes Instalados​

La lista de paquetes muestra todos los paquetes instalados con:

  • Nombre: El nombre para mostrar del paquete
  • Versión: Versión instalada actualmente
  • Descripción: Breve descripción de la funcionalidad
  • Estado: Si está habilitado o deshabilitado

Gestionando Paquetes​

Instalando Paquetes​

  1. Haz clic en el botón Instalar
  2. Selecciona un archivo de paquete (.zip o repositorio de paquetes)
  3. Confirma la instalación
  4. Espera a que la instalación se complete

Actualizando Paquetes​

Cuando hay actualizaciones disponibles:

  1. Un indicador de actualización aparece junto al paquete
  2. Haz clic en el botón Actualizar para instalar la actualización
  3. Reinicia Rayforge si se te solicita

Eliminando Paquetes​

  1. Selecciona el paquete a eliminar
  2. Haz clic en el botón Eliminar
  3. Confirma la eliminación
aviso

Eliminar un paquete puede afectar documentos que usan sus características.

Fuentes de Paquetes​

Los paquetes pueden instalarse desde:

  • Archivos locales: Archivos .zip descargados manualmente
  • Repositorios: Repositorios Git que contienen paquetes
  • Registro oficial: Paquetes curados por el equipo de Rayforge
