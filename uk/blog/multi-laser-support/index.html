Підтримка кількох лазерів - Вибирайте різні лазери для кожної операції

10 січня 2025 р. · 3 хв. читання
Rayforge Team
Core development team

Одна з найпотужніших функцій Rayforge - це можливість призначати різні лазери різним операціям в межах одного завдання. Це відкриває хвилюючі можливості для налаштувань з кількома інструментами та спеціалізованих робочих процесів.

Що таке підтримка кількох лазерів?​

Якщо ваша машина оснащена кількома лазерними модулями—скажімо, діодний лазер для гравірування та CO2 лазер для різання, або діодні лазери різної потужності, оптимізовані для різних матеріалів—Rayforge дозволяє повністю скористатися цим налаштуванням.

З підтримкою кількох лазерів ви можете:

  • Призначати різні лазери різним операціям у вашому завданні
  • Перемикатися між лазерними модулями автоматично під час виконання завдання
  • Оптимізувати для матеріалу та завдання використовуючи правильний інструмент для кожної операції

Випадки використання​

Гібридне гравірування та різання​

Уявіть, що ви працюєте над проектом дерев'яної вивіски:

  1. Операція 1: Використовуйте низькопотужний діодний лазер для гравірування дрібного тексту та детальної графіки
  2. Операція 2: Перемкніться на більш потужний CO2 лазер щоб вирізати форму вивіски

З Rayforge ви просто призначаєте кожну операцію відповідному лазеру у вашому профілі машини, і програмне забезпечення обробляє решту.

Оптимізація для конкретних матеріалів​

Різні типи лазерів відмінно працюють з різними матеріалами:

  • Діодні лазері: Чудові для гравірування по дереву, шкіри та деяких пластиків
  • CO2 лазері: Відмінні для різання акрилу, дерева та інших органічних матеріалів
  • Волоконні лазері: Ідеальні для маркування металів

Якщо у вас є кілька типів лазерів на гантрі-системі, підтримка кількох лазерів Rayforge дозволяє використовувати оптимальний інструмент для кожної частини вашого проекту.

Як це налаштувати​

1. Налаштуйте кілька лазерів у вашому профілі машини​

Перейдіть до Налаштування машини → Кілька лазерів та визначте кожен лазерний модуль у вашій системі. Ви можете вказати:

  • Тип лазера та діапазон потужності
  • Позиції зміщення (якщо лазери змонтовані в різних місцях)
  • Сумісність з матеріалами

Див. наш Посібник з конфігурації лазера для детальних інструкцій.

2. Призначте лазери операціям​

При створенні операцій у вашому проекті:

  1. Виберіть операцію (Contour, Raster тощо.)
  2. У налаштуваннях операції виберіть який лазер використовувати з випадаючого списку
  3. Налаштуйте параметри операції специфічні для цього лазера

3. Попередній перегляд та запуск​

Використовуйте 3D попередній перегляд щоб перевірити ваші інструментальні шляхи, потім надішліть завдання на вашу машину. Rayforge автоматично згенерує відповідні G-code команди для перемикання між лазерами за потреби.

Технічні деталі​

Під капотом Rayforge використовує G-code команди для перемикання між лазерними модулями. Точна реалізація залежить від вашої прошивки та апаратного налаштування, але поширені підходи включають:

  • M3/M4 з offsets інструменту: Перемикання між лазерами використовуючи команди зміни інструменту
  • GPIO керування: Використання підтримуваних прошивкою GPIO пінів для увімкнення/вимкнення різних лазерних модулів
  • Власні макроси: Визначення макросів до та після операцій для перемикання лазера

Початок роботи​

Підтримка кількох лазерів доступна в Rayforge 0.15 та пізніших версіях. Щоб почати:

  1. Оновіться до останньої версії
  2. Налаштуйте ваш профіль машини з кількома лазерами
  3. Спробуйте на тестовому проекті!

Перегляньте документацію профілів машин для деталей.

Маєте налаштування з кількома лазерами? Ми б хотіли почути про ваш досвід! Поділіться вашими проектами та відгуками на GitHub.
