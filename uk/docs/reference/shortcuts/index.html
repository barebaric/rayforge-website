Клавіатурні скорочення

Повний довідник клавіатурних скорочень у Rayforge.

:::примітка Платформні домовленості

  • Linux/Windows: клавіша Ctrl
  • macOS: клавіша  (Command)
  • Документація використовує Ctrl - користувачі macOS замінюють на  :::

Швидкий довідник за категоріями​

Найбільш використовувані (Топ 10)​

  1. Ctrl+Z / Ctrl+Y - Скасувати/Повторити
  2. Ctrl+C / Ctrl+V - Копіювати/Вставити
  3. Ctrl+D - Дублювати
  4. Delete - Видалити
  5. Ctrl+A - Вибрати все
  6. Ctrl+N - Новий ескіз
  7. Ctrl+O - Відкрити
  8. Ctrl+S - Зберегти
  9. H - Перемкнути деталі
  10. F12 - 3D вигляд

Вигляд та візуалізація​

  • H - Приховати/показати деталі
  • T - Приховати/показати вкладки
  • Alt+C - Перемкнути камеру
  • F12 - 3D вигляд
  • F11 - Режим симуляції
  • 1, 2, 7 - Пресети 3D вигляду
  • P - Перемкнути перспективу

Редагування та трансформація​

  • Ctrl+Z / Ctrl+Y - Скасувати/Повторити
  • Ctrl+X / Ctrl+C / Ctrl+V - Вирізати/Копіювати/Вставити
  • Ctrl+D - Дублювати
  • Delete - Видалити
  • Shift+H / Shift+V - Віддзеркалити Г/В
  • Ctrl+G / Ctrl+U - Згрупувати/Розгрупувати

Вирівнювання​

  • Shift+Стрілки` - Вирівняти до країв
  • Shift+Home / Shift+End - Вирівняти по центру
  • Ctrl+Shift+H / Ctrl+Shift+V - Розподілити
  • Alt+A - Піксельно ідеально

Налаштування скорочень​

:::примітка Власні скорочення Клавіатурні скорочення наразі жорстко закодовані. Конфігурація власних скорочень може бути додана в майбутніх версіях. :::

Поточні обмеження:

  • Скорочення не можуть бути змінені без модифікації вихідного коду
  • Немає GUI для налаштування скорочень
  • Деякі дії можуть не мати призначених скорочень

Запит функції: Якщо вам потрібні власні скорочення, будь ласка, відкрийте issue на GitHub.

Поради та хитрощі​

Поради щодо ефективності​

  1. Спочатку вивчіть скорочення редагування - Ctrl+Z/Y/C/V/D використовуються постійно
  2. Використовуйте одинарні клавіші перемикання - H, T, P для швидкої зміни вигляду
  3. Скорочення 3D вигляду - 1, 2, 7 для миттєвого перемикання вигляду
  4. Скорочення вирівнювання - Shift+Стрілки` швидше, ніж натискання кнопок вирівнювання
  5. Режим симуляції - Ctrl+Shift+S щоб швидко перевірити виконання

Робочі процеси з скороченнями​

Швидкий цикл редагування:

1. Новий ескіз (Ctrl+N) або Відкрити (Ctrl+O)
2. Розташувати елементи (стрілки, Shift+Стрілки для вирівнювання)
3. Дублювати частини (Ctrl+D)
4. Перевірити в 3D (F12)
5. Симулювати (F11)
6. Зберегти (Ctrl+S) або Експортувати (Ctrl+E)

Точне позиціонування:

1. Вибрати елемент
2. Стрілки для штовхань на 1мм
3. Shift+Стрілки для стрибків на 10мм
4. Shift+Home/End для центрування

Приховані скорочення​

Деякі менш відомі скорочення:

  • Ctrl+, - Швидкий доступ до налаштувань (стандарт на macOS, працює і тут)
  • Ctrl+Shift+Z - Альтернативне повторення (для користувачів, які віддають перевагу цьому замість Ctrl+Y)
  • Alt+A - Піксельно-ідеальне прилипання для точного розміщення
  • Alt+T - Швидке рівновіддалене розміщення вкладок

Платформні відмінності​

Linux​

  • Використовує стандартний модифікатор Ctrl
  • Всі скорочення працюють як документовано
  • Стандартні скорочення GTK застосовуються в текстових полях

Windows​

  • Використовує стандартний модифікатор Ctrl
  • Ідентичні скороченням Linux
  • Специфічні для Windows клавіші (Win клавіша) не використовуються

macOS​

  • Ctrl відображається на  (Command)
  • Alt відображається на % (Option)
  • Застосовуються стандартні домовленості macOS
  • Cmd+Q щоб вийти (замість Ctrl+Q)

Виправлення проблем зі скороченнями​

Скорочення не працює​

Поширені проблеми:

  1. Фокус на неправильному елементі - Переконайтеся, що полотно або головне вікно має фокус, а не текстове поле
  2. Конфліктуючий додаток - Інший додаток може перехоплювати скорочення
  3. Скорочення робочого середовища - Системні скорочення можуть перекривати (наприклад, Alt+F4)
  4. Numpad проти числового ряду - Використовуйте числовий ряд для скорочень 3D вигляду, не numpad

Рішення:

  • Натисніть на полотно, щоб переконатися, що воно має фокус
  • Перевірте системні клавіатурні скорочення на конфлікти
  • Спробуйте дію з меню натомість, щоб перевірити функціональність

Модифікаторні клавіші​

Якщо Ctrl не працює:

  • Перевірте, що Caps Lock вимкнено (може заважати на деяких системах)
  • Спробуйте Ctrl з обох сторін клавіатури
  • Перевірте налаштування розкладки клавіатури

Якщо Alt не працює:

  • Деякі віконні менеджери захоплюють Alt для перетягування вікон
  • Спробуйте вимкнути скорочення Alt віконного менеджера
  • Використовуйте елементи меню як альтернативу
