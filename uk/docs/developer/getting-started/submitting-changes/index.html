  • –Ю–љ–Њ–≤–ї—О–є—В–µ –і–Њ–Ї—Г–Љ–µ–љ—В–∞—Ж—Ц—О –Ј–∞ –њ–Њ—В—А–µ–±–Є

–Я—А–Њ—В–µ—Б—В—Г–є—В–µ –≤–∞—И—Ц –Ј–Љ—Ц–љ–ЄвАЛ

    • –Я–Њ—Б–Є–ї–∞–љ–љ—П–Љ –љ–∞ –±—Г–і—М-—П–Ї—Ц –њ–Њ–≤'—П–Ј–∞–љ—Ц issues
    • –°–Ї—А—Ц–љ—И–Њ—В–∞–Љ–Є —П–Ї—Й–Њ –Ј–Љ—Ц–љ–∞ –≤–њ–ї–Є–≤–∞—Ф –љ–∞ UI

  • –Ю–њ–µ—А–∞—В–Є–≤–љ–Њ —А–µ–∞–≥—Г–є—В–µ –љ–∞ –≤—Ц–і–≥—Г–Ї–Є —В–∞ –≤–љ–Њ—Б—М—В–µ –Ј–∞–њ–Є—В—Г–≤–∞–љ—Ц –Ј–Љ—Ц–љ–Є
  • –Ґ—А–Є–Љ–∞–є—В–µ –і–Є—Б–Ї—Г—Б—Ц—О —Б—Д–Њ–Ї—Г—Б–Њ–≤–∞–љ–Њ—О —В–∞ –Ї–Њ–љ—Б—В—А—Г–Ї—В–Є–≤–љ–Њ—О
