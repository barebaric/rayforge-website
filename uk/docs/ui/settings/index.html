Налаштування

Налаштуйте Rayforge відповідно до вашого робочого процесу та вподобань.

Доступ до налаштувань​

  • Меню: Редагування → Налаштування
  • Клавіатура: ctrl+кома

Загальні​

Сторінка Загальні містить налаштування програми.

Зовнішній вигляд​

  • Тема: Виберіть між Системною, Світлою або Темною темою

Одиниці​

Налаштуйте одиниці відображення для різних значень у програмі:

  • Довжина: Міліметри, дюйми та інші одиниці довжини
  • Швидкість: мм/хв, мм/сек, дюйми/хв та інші одиниці швидкості
  • Прискорення: мм/с² та інші одиниці прискорення

Запуск​

Налаштуйте, що відбувається при запуску програми. Файли, вказані в командному рядку, завжди перекривають ці налаштування.

  • Поведінка при запуску:
    • Нічого не відкривати
    • Відкрити останній проект
    • Відкрити конкретний проект
  • Шлях до проекту: Шлях до конкретного проекту для відкриття при запуску (видно тільки коли вибрано "Відкрити конкретний проект")

Машини​

Сторінка Машини дозволяє керувати конфігураціями вашої машини.

  • Додати машину: Створити нову конфігурацію машини
  • Видалити машину: Видалити вибрану машину
  • Список машин: Показує всі налаштовані машини з їх статусом з'єднання

Для детальної конфігурації машини дивіться розділ Налаштування машини.

Матеріали​

Сторінка Матеріали керує вашою бібліотекою матеріалів.

  • Додати матеріал: Створити новий пресет матеріалу
  • Імпортувати бібліотеку: Імпортувати матеріали з файлу
  • Експортувати бібліотеку: Експортувати ваші матеріали для резервного копіювання або обміну

Дивіться Матеріали для отримання детальнішої інформації.

Рецепти​

Сторінка Рецепти керує вашими рецептами операцій.

  • Додати рецепт: Створити новий рецепт
  • Імпортувати рецепти: Імпортувати рецепти з файлу
  • Експортувати рецепти: Експортувати ваші рецепти для резервного копіювання або обміну

Дивіться Рецепти для отримання детальнішої інформації.

Аддони​

Сторінка Аддони показує встановлені аддони розширень.

  • Список аддонів: Показує всі встановлені аддони
  • Деталі аддону: Перегляд інформації про кожен аддон
