Налаштування обладнання

Сторінка "Обладнання" у налаштуваннях машини налаштовує фізичні розміри, систему координат та обмеження руху вашої машини.

Осі​

Налаштуйте діапазони осей та систему координат для вашої машини.

Діапазон X/Y​

Повний хід кожної осі в одиницях машини.

  • Виміряйте фактичну область різання, а не зовнішній розмір машини
  • Врахуйте будь-які перешкоди або обмеження
  • Приклад: 400 для типового лазера K40

Початок координат​

Виберіть, де знаходиться початок координат (0,0) вашої машини. Це визначає, як інтерпретуються координати.

  • Лівий нижній: Найпоширеніший для пристроїв GRBL. X збільшується вправо, Y збільшується вгору.
  • Лівий верхній: Поширений для деяких машин типу CNC. X збільшується вправо, Y збільшується вниз.
  • Правий верхній: X збільшується вліво, Y збільшується вниз.
  • Правий нижній: X збільшується вліво, Y збільшується вгору.

Визначення вашого початку координат​

  1. Виконайте回家 (homing) вашої машини за допомогою кнопки Home
  2. Спостерігайте, куди переміщується лазерна головка
  3. Ця позиція є вашим початком координат (0,0)

:::інформація Налаштування початку координат впливає на те, як генерується G-code. Переконайтеся, що воно відповідає конфігурації homing у вашій прошивці. :::

Напрямок осі​

Змініть напрямок будь-якої осі за потреби:

  • Інвертувати вісь X: Робить значення координат X від'ємними
  • Інвертувати вісь Y: Робить значення координат Y від'ємними
  • Інвертувати вісь Z: Увімкніть, якщо позитивна команда Z (наприклад, G0 Z10) переміщує головку вниз

Робоча область​

Поля визначають непридатний простір навколо країв діапазонів осей. Це корисно, коли ваша машина має ділянки, куди лазер не може дістати (наприклад, через складальну частину лазерної головки, кабельні ланцюги або інші перешкоди).

  • Ліве/Верхнє/Праве/Нижнє поле: Непридатний простір від кожного краю в одиницях машини

Коли поля встановлені, робоча область (придатний простір) розраховується як діапазони осей мінус поля.

Програмні обмеження​

Налаштовані межі безпеки для переміщення головки машини. Коли увімкнено, елементи керування переміщенням запобігатимуть руху за межами цих обмежень.

  • Увімкнути власні програмні обмеження: Перемикач для використання власних обмежень замість меж робочої поверхні
  • X/Y Мін: Мінімальна координата для кожної осі
  • X/Y Макс: Максимальна координата для кожної осі

Програмні обмеження автоматично обмежуються, щоб залишатися в межах діапазонів осей (від 0 до значення діапазону).

:::порада Використовуйте програмні обмеження для захисту ділянок вашої робочої поверхні, які ніколи не повинні досягатися під час переміщення, наприклад, ділянок з оснащенням або чутливим обладнанням. :::
