Компенсація керфа

Керф — це матеріал, видалений лазерним променем під час різання. Компенсація керфа коригує траєкторії інструменту з урахуванням цього, гарантуючи, що вирізані деталі відповідають своїм проектним розмірам.

Що таке керф?​

Керф = ширина матеріалу, видаленого під час процесу різання.

Приклад:

  • Розмір плями лазера: 0.2мм
  • Взаємодія з матеріалом: додає ~0.1мм з кожного боку
  • Загальний керф: ~0.4мм

Як працює компенсація керфа​

Компенсація керфа зміщує траєкторію інструменту всередину або назовні для врахування видалення матеріалу:

Для зовнішніх різів (різання деталі):

  • Зміщує траєкторію назовні на половину ширини керфа
  • Результат: Кінцева деталь правильного розміру

Для внутрішніх різів (різання отвору):

  • Зміщує траєкторію всередину на половину ширини керфа
  • Результат: Кінцевий отвір правильного розміру

Приклад з керфом 0.4мм:

Оригінальна траєкторія:  квадрат 50мм
Компенсація:             Зміщення назовні на 0.2мм (половина керфа)
Лазер слідує:            квадрат 50.4мм
Після різання:           Деталь вимірює 50.0мм (ідеально!)

Вимірювання керфа​

Процедура точного вимірювання керфа:

  1. Створіть тестовий файл:

    • Намалюйте квадрат 50мм x 50мм
    • Намалюйте коло (будь-який розмір, для тесту внутрішнього різу)

  2. Виріжте тест:

    • Використовуйте ваші звичайні налаштування різання
    • Ріжте повністю наскрізь
    • Дайте матеріалу охолонути

  3. Виміряйте:

    • Зовнішній квадрат (деталь): Виміряйте штангенциркулем
      • Якщо < 50мм, керф був видалений назовні
      • Керф = (50 - виміряне) x 2
    • Внутрішнє коло (отвір): Виміряйте діаметр
      • Якщо > проектного діаметра, керф був видалений всередину
      • Керф = (виміряне - проектне) / 2

  4. Усередніть: Використовуйте середнє з кількох вимірювань

Змінні, що впливають на керф:

  • Потужність лазера (вища = ширший)
  • Швидкість різання (повільніше = ширший)
  • Тип та щільність матеріалу
  • Відстань фокусування
  • Тиск повітряного ассистента

Ручна компенсація керфа​

Якщо автоматична компенсація керфа недоступна, компенсуйте у вашому дизайнерському ПЗ:

Inkscape:

  1. Виберіть траєкторію
  2. Шлях → Динамічне зміщення (Ctrl+J)
  3. Перетягніть для зміщення на половину вашого вимірювання керфа
    • Назовні для деталей (щоб зробити траєкторію більшою)
    • Всередину для отворів (щоб зробити траєкторію меншою)
  4. Шлях → Об'єкт у шлях для завершення

Illustrator:

  1. Виберіть траєкторію
  2. Об'єкт → Шлях → Зміщення шляху
  3. Введіть значення зміщення: (керф / 2)
    • Позитивне назовні, негативне всередину
  4. OK для застосування

Fusion 360 / CAD:

  • Зміщуйте елементи ескізу перед експортом
  • Використовуйте розмір керфа/зміщення
