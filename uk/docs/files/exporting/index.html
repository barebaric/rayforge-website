Експорт з Rayforge

Rayforge підтримує кілька варіантів експорту для різних цілей:

  • G-code - Вивід керування машиною для запуску завдань
  • Експорт об'єктів - Експорт окремих деталей у векторні формати
  • Експорт документа - Експорт всіх деталей як одного файлу

Експорт об'єктів​

Ви можете експортувати будь-яку деталь у векторні формати для використання у дизайнерському ПЗ, CAD- додатках або для архівування.

Як експортувати​

  1. Виберіть деталь на полотні
  2. Виберіть Об'єкт → Експортувати об'єкт... (або правою кнопкою → Експортувати об'єкт...)
  3. Виберіть формат та місце збереження

Доступні формати​

ФорматРозширенняОпис
RFS.rfsВласний параметричний формат ескізів Rayforge. Зберігає всі обмеження і може бути реімпортований для редагування.
SVG.svgScalable Vector Graphics. Широко сумісний з дизайнерським ПЗ, як Inkscape, Illustrator, та веб-браузери.
DXF.dxfDrawing Exchange Format. Сумісний з більшістю CAD-додатків, як AutoCAD, FreeCAD, та LibreCAD.

Примітки до експорту​

  • SVG та DXF експортують розв'язану геометрію (не параметричні обмеження)
  • Експорт використовує міліметрові одиниці
  • Геометрія масштабується до фактичних розмірів (світовий простір)
  • Кілька підшляхів (роз'єднані форми) зберігаються як окремі елементи

Випадки використання​

Спільне використання дизайнів:

  • Експорт у SVG для спільного використання з користувачами Inkscape
  • Експорт у DXF для користувачів CAD-ПЗ

Подальше редагування:

  • Експорт у SVG/DXF, редагування у зовнішньому ПЗ, реімпорт

Архівування:

  • Використовуйте RFS для дизайнів на основі ескізів для збереження редагованості
  • Використовуйте SVG/DXF для довготривалого зберігання або для користувачів без Rayforge

Експорт документів​

Ви можете експортувати всі деталі в документі в один векторний файл. Це корисно для спільного використання повних проектів або створення резервних копій у стандартних форматах.

Як експортувати​

  1. Виберіть Файл → Експортувати документ...
  2. Виберіть формат (SVG або DXF)
  3. Виберіть місце збереження

Доступні формати​

ФорматРозширенняОпис
SVG.svgScalable Vector Graphics. Широко сумісний з дизайнерським ПЗ, як Inkscape, Illustrator, та веб-браузери.
DXF.dxfDrawing Exchange Format. Сумісний з більшістю CAD-додатків, як AutoCAD, FreeCAD, та LibreCAD.

Примітки до експорту​

  • Всі деталі з усіх шарів об'єднуються в один файл
  • Позиції деталей зберігаються
  • Порожні деталі пропускаються
  • Обмежувальний прямокутник охоплює всю геометрію

Випадки використання​

  • Спільне використання проекту: Експорт всього проекту для співпраці
  • Резервне копіювання: Створення візуального архіву вашої роботи
  • Подальше редагування: Перенесення всього дизайну в Inkscape або CAD-ПЗ

Експорт G-code​

Згенерований G-code містить все точно так, як це буде надіслано машині. Точний формат, команди, числова точність тощо залежать від налаштувань поточно вибраної машини та її діалекту G-code.

Методи експорту​

Метод 1: Меню "Файл"​

Файл Експортувати G-code (Ctrl+E)

  • Відкриває діалог збереження файлу
  • Виберіть розташування та ім'я файлу
  • G-code генерується та зберігається

Метод 2: Командний рядок​

# Експорт з командного рядка (якщо підтримується)
rayforge --export output.gcode input.svg

Вивід G-code​
