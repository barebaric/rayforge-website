Загальні профілі

Якщо вашого пристрою немає у списку вище, спробуйте один із загальних профілів — Generic GRBL, Generic Ruida, Generic Smoothieware. Вони працюють з будь-якою supported firmware і можуть бути налаштовані через configuration wizard.

Не вистачає пристрою?

Rayforge використовує installable device profiles — нові машини можна додавати без оновлення самого застосунку. Якщо ви хочете додати профіль для свого пристрою, перегляньте developer documentation або open an issue.
