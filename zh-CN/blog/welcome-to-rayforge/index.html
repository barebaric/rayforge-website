欢迎来到 Rayforge：现代激光控制软件

2024年12月15日 · 阅读时长约 1 分钟
Rayforge Team
Core development team

我们很高兴向您介绍 Rayforge，这是一款专为基于 GRBL 的激光切割机和雕刻机设计的现代化开源 G-code 发送器和控制软件。

为什么选择 Rayforge？​

在激光切割和雕刻的世界中，拥有可靠、直观的软件与拥有优质硬件同样重要。Rayforge 从零开始构建，专注于：

  • 现代用户体验：基于 Gtk4 和 Libadwaita 构建，Rayforge 提供简洁的原生界面，在 Linux 和 Windows 上都能完美运行
  • 专业功能：多图层工作流程、3D 预览、相机集成和高级路径生成
  • 灵活性：支持多个机器配置文件、自定义宏和丰富的配置选项
  • 开放开发：社区驱动，欢迎贡献

Rayforge 有何不同？​

多图层操作​

Rayforge 的突出功能之一是其多图层工作流程。将不同的操作分配给设计中的不同图层——在一个图层上雕刻精细细节，然后在另一个图层上切割出作品。所有操作都在一个界面中无缝管理。

3D G-code 预览​

在将作业发送到机器之前，通过我们的 3D G-code 预览精确可视化将要发生的操作。在问题发生之前发现潜在问题，节省时间和材料。

相机集成​

使用 USB 摄像头进行精确的工件对齐、定位，甚至背景描摹。非常适合在现有材料上定位设计或制作夹具。

高级路径生成​

Rayforge 包括高质量图像描摹、移动时间优化、路径平滑和光斑大小插值，确保您的项目按计划完美呈现。

开始使用​

准备好尝试 Rayforge 了吗？请查看我们的安装指南开始使用。无论您使用的是 Ubuntu、通过 Snap 安装的其他 Linux 发行版，还是 Windows，我们都能满足您的需求。

加入社区​

Rayforge 是开源的，由社区驱动。我们欢迎各种形式的贡献：

  • 在 GitHub Issues 上报告错误和请求功能
  • 通过 pull requests 贡献代码
  • 改进文档
  • 分享您的项目和经验

有问题或反馈？欢迎在 GitHub 上参与讨论或查看我们的文档。
