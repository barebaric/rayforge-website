首次设置

安装 Rayforge 后，您需要配置激光切割机或雕刻机。本指南将带您完成创建第一台机器和建立连接。

第 1 步：启动 Rayforge​

从应用程序菜单启动 Rayforge，或在终端中运行 rayforge。您应该看到带有空白画布的主界面。

第 2 步：创建机器​

导航到 设置 → 机器 或按 ctrl+comma 打开设置对话框，然后选择 机器 页面。

点击 添加机器 创建新机器。您可以：

  1. 选择内置配置文件 - 从预定义的机器模板中选择
  2. 选择"自定义" - 从空白配置开始

选择后，机器设置对话框将打开您的新机器。

第 3 步：配置常规设置​

常规 页面包含基本机器信息、驱动程序选择和连接设置。

机器信息​

  1. 机器名称：给您的机器一个描述性名称（例如，"K40 激光"、"Ortur LM2"）

驱动程序选择​

从下拉菜单中选择适合您设备的驱动程序：

  • GRBL (串口) - 用于通过 USB/串口连接的 GRBL 设备
  • GRBL (网络) - 用于具有 WiFi/以太网连接的 GRBL 设备
  • Smoothie - 用于基于 Smoothieware 的设备

驱动程序设置​

根据您选择的驱动程序，配置连接参数：

GRBL (串口) - USB​

  1. 端口：从下拉菜单中选择设备（例如，Linux 上是 /dev/ttyUSB0，Windows 上是 COM3）
  2. 波特率：选择 115200（大多数 GRBL 设备的标准）
信息

如果您的设备未出现在列表中，请检查它是否已连接以及您是否有必要的权限。在 Linux 上，您可能需要将用户添加到 dialout 组。

GRBL (网络) / Smoothie - WiFi/以太网​

  1. 主机：输入设备的 IP 地址（例如，192.168.1.100）
  2. 端口：输入端口号（通常是 23 或 8080）

速度和加速度​

这些设置用于作业时间估计和路径优化：

  • 最大空行速度：最大快速移动速度
  • 最大切割速度：最大切割速度
  • 加速度：用于时间估计和过扫描计算

第 4 步：配置硬件设置​

切换到 硬件 选项卡以设置机器的物理尺寸。

尺寸​

  • 宽度：输入工作区域的最大宽度（毫米）
  • 高度：输入工作区域的最大高度（毫米）

轴​

  • 坐标原点 (0,0)：选择机器原点的位置：
    • 左下角（GRBL 最常见）
    • 左上角
    • 右上角
    • 右下角

轴偏移（可选）​

如果您的机器需要精确偏移，配置 X 和 Y 偏移。

第 5 步：自动连接​

Rayforge 在应用程序启动时自动连接到您的机器（如果机器已开机并连接）。您不需要手动点击连接按钮。

连接状态显示在主窗口左下角，带有状态图标和标签，显示当前状态（已连接、正在连接、已断开、错误等）。

已连接！

如果您的机器显示"已连接"状态，您就可以开始使用 Rayforge 了！

可选：高级配置​

多激光器​

如果您的机器有多个激光模块（例如，二极管和 CO2），您可以在 激光 页面中配置它们。

有关详细信息，请参阅激光配置。

相机设置​

如果您有用于对齐和定位的 USB 相机，请在 相机 页面中配置它。

有关详细信息，请参阅相机集成。

设备设置​

设备 页面允许您直接读取和修改连接设备上的固件设置（如 GRBL 参数）。这是一个高级功能，应谨慎使用。

注意

编辑设备设置可能是危险的，如果应用不正确的值，可能会导致机器无法操作！

连接问题故障排除​

找不到设备​

  • Linux（串口）：将用户添加到 dialout 组。对于基于 Debian 的发行版上的 Snap 和非 Snap 安装，都需要此操作以避免 AppArmor DENIED 消息：

    sudo usermod -a -G dialout $USER

    注销并重新登录以使更改生效。

  • Snap 包：除了上述 dialout 组外，还需确保您已授予串口权限：

    sudo snap connect rayforge:serial-port

  • Windows：检查设备管理器以确认设备被识别并记下 COM 端口号。

连接被拒绝​

  • 验证 IP 地址和端口号是否正确
  • 确保机器已开机并连接到网络
  • 如果使用网络连接，检查防火墙设置

机器无响应​

  • 尝试不同的波特率（有些设备使用 9600 或 57600）
  • 检查电缆是否松动或连接不良
  • 重启激光切割机并重试

有关更多帮助，请参阅连接问题。
