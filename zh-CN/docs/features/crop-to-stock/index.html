裁剪到材料

裁剪到材料将切割路径限制在材料边界内。任何延伸到材料区域之外的切割都会被修剪，防止激光在材料外切割。

工作原理​

变换器将您的切割路径与定义的材料边界进行比较。此边界之外的路径段被移除或裁剪到材料边缘。

如果文档中没有定义材料项，裁剪边界将回退到机器的工作区域。

设置​

启用裁剪到材料​

开启或关闭裁剪。默认禁用。

偏移​

在裁剪前调整有效的材料边界（-100 到 +100 mm）。

  • 正值： 缩小边界（更保守地切割）
  • 负值： 扩大边界（允许更靠近边缘切割）
  • 0 mm： 使用精确的材料边界

当您希望与材料边缘有安全边距，或当您的材料放置未完全对齐时，使用偏移。

何时使用裁剪到材料​

部分设计： 您的设计大于您的材料，但您只想切割适合的部分。

安全边距： 防止意外切割超出材料边缘。

嵌套板材： 仅切割适合当前材料块的部分。

测试切割： 将测试限制在材料的特定区域。

示例​

您有一个大设计但只有一小块材料：

  1. 定义您的材料尺寸以匹配您的材料
  2. 启用裁剪到材料
  3. 将偏移设置为 2mm 以获得安全边距
  4. 仅材料边界内的部分将被切割
