多层工作流

Rayforge 的多层系统允许您将复杂的任务组织为独立的处理阶段，每个阶段都有 自己的操作和设置。这对于组合不同工艺（如雕刻和切割）或处理多种材料至关 重要。

什么是层？​

Rayforge 中的层是：

  • 容器，用于存放工件（导入的形状、图像、文本）
  • 工作流，定义如何处理这些工件
  • 步骤，在任务执行期间按顺序处理

核心概念： 层按顺序处理，一个接一个，让您可以控制操作的执行顺序。

层与工件

一个层包含一个或多个工件。导入带有层的 SVG 文件时，设计中的每个层都会 成为 Rayforge 中的独立层。这让您可以按照创建时的方式保持设计的组织结构。

为什么要使用多层？​

常见用例​

1. 先雕刻再切割

最常见的多层工作流：

  • 第 1 层： 光栅雕刻设计图案
  • 第 2 层： 轮廓切割外框

为什么要分层？

  • 先雕刻确保工件在雕刻过程中不会移动
  • 后切割防止工件在雕刻完成前掉落
  • 每个操作使用不同的功率/速度设置

2. 多次切割

对于厚材料：

  • 第 1 层： 中等功率的第一刀
  • 第 2 层： 全功率的第二刀（相同几何图形）
  • 第 3 层： 可选的第三刀（如需要）

优势：

  • 与单次高功率切割相比，减少烧焦
  • 每层可以有不同的速度/功率设置

3. 多材料项目

一个任务中使用不同材料：

  • 第 1 层： 切割亚克力部件
  • 第 2 层： 雕刻木质部件
  • 第 3 层： 标记金属部件

要求：

  • 每层针对工作台的不同区域
  • 每种材料使用不同的速度/功率/焦点

4. SVG 图层导入

导入具有现有图层结构的 SVG 文件：

  • 第 1 层： 来自 SVG 的雕刻元素
  • 第 2 层： 来自 SVG 的切割元素
  • 第 3 层： 来自 SVG 的划线元素

工作流程：

  • 导入包含图层的 SVG 文件
  • 在导入对话框中启用"使用原始矢量"
  • 从检测到的图层列表中选择要导入的图层
  • 每个图层成为 Rayforge 中的独立层

要求：

  • 您的 SVG 文件必须使用图层（在 Inkscape 或类似软件中创建）
  • 导入时启用"使用原始矢量"
  • 图层名称从您的设计软件中保留

创建和管理层​

添加新层​

  1. 点击"+"按钮 在图层面板中
  2. 为图层命名，使用描述性名称（如"雕刻层"、"切割层"）
  3. 图层出现 在图层列表中

默认： 新文档从一层开始。

图层属性​

每个层具有：

属性描述
名称在图层列表中显示的名称
可见性切换画布和预览中的可见性
工作流应用于该层工件的操作
旋转该层是否在旋转模式下运行
工件该层中包含的形状/图像

按层设置旋转模式​

如果您配置了旋转附件，可以为各个层启用旋转模式。 这允许您在同一个项目中混合平面工作和圆柱工作 — 例如，在一个层上雕刻 盒子平面盖板上的图案，在另一个层上在圆柱体表面环绕文字。

启用旋转模式的层在图层列表中显示一个小旋转图标。每个层记住自己的旋转 设置，您可以自由混合搭配。

层作为容器

层是工件的容器。导入带有图层的 SVG 文件时，设计中的每个图层都会成为 Rayforge 中的独立层。

重新排列层​

执行顺序 = 图层列表中的顺序（从上到下）

重新排列方法：

  1. 拖放 图层面板中的层
  2. 顺序很重要 - 层从上到下执行

示例：

图层面板：
1. 雕刻层     最先执行
2. 划线层     第二执行
3. 切割层     最后执行（推荐）

删除层​

  1. 选择图层 在图层面板中
  2. 点击删除按钮 或按 Delete 键
  3. 确认删除（该层中的所有工件将被移除）
删除是不可恢复的

删除图层会移除其所有工件和工作流设置。如果误删，请使用撤销功能。

将工件分配到层​

手动分配​

  1. 导入或创建 一个工件
  2. 将工件拖到 图层面板中的目标层
  3. 或使用属性面板 更改工件所属的层

SVG 图层导入​

启用"使用原始矢量"导入 SVG 文件时：

  1. 启用"使用原始矢量" 在导入对话框中
  2. Rayforge 检测图层 来自您的 SVG 文件
  3. 选择要导入的图层 使用图层开关
  4. 每个选中的图层 成为独立层，带有自己的工件
图层检测

Rayforge 自动检测 SVG 文件中的图层。您在设计软件中创建的每个图层都 将显示为 Rayforge 中的独立层。

仅限矢量导入

图层选择仅在使用直接矢量导入时可用。使用追踪模式时，整个 SVG 作为 一个工件处理。

在层之间移动工件​

拖放：

  • 在画布或文档面板中选择工件
  • 拖动到图层面板中的目标层

剪切和粘贴：

  • 从当前层剪切工件（Ctrl+X）
  • 选择目标层
  • 粘贴（Ctrl+V）

SVG 导入对话框​

导入 SVG 文件时，导入对话框提供影响图层处理的选项：

导入模式：

  • 使用原始矢量： 保留您的矢量路径和图层结构。启用后，会出现 "图层"部分，显示文件中的所有图层。
  • 追踪模式： 将 SVG 转换为位图并追踪轮廓。此模式下图层选择 被禁用。

图层部分（仅限矢量导入）：

  • 显示 SVG 文件中的所有图层
  • 每个图层有开关来启用/禁用导入
  • 设计软件中的图层名称被保留
  • 只有选中的图层被导入为独立层
准备 SVG 文件以进行图层导入

要使用 SVG 图层导入，请在 Inkscape 等软件中使用图层创建设计。使用 图层面板组织您的设计，Rayforge 将保留该结构。

图层工作流​

每个层都有一个工作流，定义其工件如何被处理。

设置图层工作流​

对于每个层，您选择操作类型并配置其设置：

操作类型：

  • 轮廓 - 沿轮廓线行进（用于切割或划线）
  • 光栅雕刻 - 雕刻图像和填充区域
  • 深度雕刻 - 创建深度可变的雕刻

可选增强功能：

  • 固定桥 - 切割时固定零件的小桥接
  • 过扫描 - 将切割延伸到形状之外以获得更干净的边缘
  • 切缝补偿 - 补偿激光的切割宽度

常见图层配置​

雕刻层：

  • 操作：光栅雕刻
  • 设置：300-500 DPI，中等速度
  • 通常不需要额外选项

切割层：

  • 操作：轮廓切割
  • 选项：固定桥（固定零件），过扫描（干净边缘）
  • 设置：较慢速度，较高功率

划线层：

  • 操作：轮廓（轻微功率，不切穿）
  • 设置：低功率，快速度
  • 用途：折线，装饰线

图层可见性​

控制哪些层在画布和预览中显示：

画布可见性​

  • 眼睛图标 在图层面板中切换可见性
  • 隐藏的层：
    • 不在 2D 画布中显示
    • 不在 3D 预览中显示
    • 仍然包含在生成的 G-code 中

使用场景：

  • 在定位切割层时隐藏复杂的雕刻层
  • 处理特定层时整理画布
  • 一次专注于一个层

可见性 vs 启用​

状态画布预览G-code
可见且已启用是是是
隐藏且已启用否否是
可见但未启用是是否
隐藏且未启用否否否
禁用图层

要临时将层从任务中排除而不删除它，请关闭该层的操作或在图层设置中 禁用它。

图层执行顺序​

层如何被处理​

在任务执行期间，Rayforge 从上到下按顺序处理每个层。在每个层内，所有 工件在移至下一层之前被处理。

顺序很重要​

错误顺序：

1. 切割层
2. 雕刻层

问题： 切割后的零件可能在雕刻前掉落或移位！

正确顺序：

1. 雕刻层
2. 切割层

原因： 雕刻在零件仍然固定时进行，然后切割将其分离。

多次切割​

对于厚材料，创建多个切割层：

1. 雕刻层
2. 切割层（第 1 刀） - 50% 功率
3. 切割层（第 2 刀） - 75% 功率
4. 切割层（第 3 刀） - 100% 功率

提示： 对所有切割刀使用相同的几何图形（复制图层）。

高级技巧​

按材料分组图层​

运行混合任务时，使用图层按材料组织：

材料 1（3mm 亚克力）：
  - 亚克力雕刻层
  - 亚克力切割层

材料 2（3mm 胶合板）：
  - 木材雕刻层
  - 木材切割层

工作流程：

  1. 处理所有材料 1 的层
  2. 更换材料
  3. 处理所有材料 2 的层

替代方案： 对不同材料使用单独的文档。

在层之间暂停​

您可以配置 Rayforge 在层之间暂停。这在以下情况下很有用：

  • 任务中途更换材料
  • 在继续之前检查进度
  • 为不同操作调整焦点

要设置层间暂停，请使用机器设置中的钩子功能。

各层的特定设置​

每个层的工作流可以有独特的设置：

层操作速度功率次数
雕刻光栅300 mm/min20%1
划线轮廓500 mm/min10%1
切割轮廓100 mm/min90%2

最佳实践​

命名约定​

好的图层名称：

  • "雕刻 - Logo"
  • "切割 - 外轮廓"
  • "划线 - 折线"
  • "第 1 刀 - 粗切"
  • "第 2 刀 - 精切"

不好的图层名称：

  • "层 1"、"层 2"（不具描述性）
  • 过长的描述（保持简洁）

图层组织​

  1. 从上到下 = 执行顺序
  2. 先雕刻后切割（一般规则）
  3. 将相关操作分组（所有切割、所有雕刻）
  4. 使用可见性 专注于当前工作
  5. 删除未使用的层 以保持项目整洁

准备 SVG 文件以进行图层导入​

为获得最佳 SVG 图层导入效果：

  1. 使用图层面板 在您的设计软件中组织设计
  2. 为每个图层指定有意义的名称（如"雕刻"、"切割"）
  3. 保持图层扁平 - 避免将图层放在其他图层内
  4. 保存文件 并导入 Rayforge
  5. 验证图层检测 通过检查导入对话框

Rayforge 与在 Inkscape 或支持图层的类似矢量设计软件中创建的 SVG 文件配合使用效果最佳。

性能​

多层：

  • 对性能没有显著影响
  • 10-20 层在复杂任务中很常见
  • 按逻辑组织，而非为了减少图层数量

必要时简化：

  • 尽可能将相似操作合并到一个层中
  • 减少光栅雕刻数量（最耗费资源）

故障排除​

图层未生成 G-code​

问题： 图层出现在文档中但未出现在生成的 G-code 中。

解决方案：

  1. 检查图层是否有工件 - 空层被跳过
  2. 检查工作流是否已配置 - 图层需要一个操作
  3. 验证操作设置 - 功率 > 0，有效速度等
  4. 检查工件可见性 - 隐藏的工件可能不被处理
  5. 重新生成 G-code - 做一个小改动强制重新生成

图层顺序错误​

问题： 操作以意外顺序执行。

解决方案： 在图层面板中重新排列图层。记住：顶部 = 最先执行。

图层在预览中重叠​

问题： 多个层在预览中显示重叠内容。

说明： 如果层共享相同的 XY 区域，这是正常的。

解决方案：

  • 使用图层可见性临时隐藏其他层
  • 检查 3D 预览以查看深度/顺序
  • 确认这是有意的（如先雕刻再切割同一形状）

工件在错误的层中​

问题： 工件被分配到了错误的层。

解决方案： 将工件拖到图层面板或文档树中的正确层。

SVG 图层未检测到​

问题： 导入 SVG 文件但导入对话框中没有出现图层。

解决方案：

  1. 检查 SVG 结构 - 在 Inkscape 或类似软件中打开文件以验证其 包含图层
  2. 启用"使用原始矢量" - 图层选择仅在此导入模式下可用
  3. 确认您的设计有图层 - 确保您在设计软件中创建的是图层而不 仅是分组
  4. 检查嵌套图层 - 其他图层内的图层可能无法正确检测
  5. 重新保存文件 - 有时用设计软件的当前版本重新保存会有帮助

SVG 图层导入显示错误内容​

问题： 导入的图层显示其他图层的内容或为空。

解决方案：

  1. 检查图层选择 - 确认导入对话框中启用了正确的图层
  2. 验证您的设计 - 在设计软件中打开原始文件确认每个图层包含 正确内容
  3. 检查共享元素 - 出现在多个图层中的元素可能导致混淆
  4. 尝试追踪模式 - 如果矢量导入有问题，使用追踪模式作为备选
