过扫描

过扫描将光栅雕刻线延伸到实际内容区域之外，以确保激光在雕刻期间达到恒定速度，消除加速伪影。

问题：加速标记​

没有过扫描，光栅雕刻会遭受加速伪影：

  • 浅色边缘，加速度开始的地方（激光移动太快于功率水平）
  • 深色边缘，减速度发生的地方（激光停留更长）
  • 整条线的雕刻深度/暗度不一致
  • 线边缘可见条带或条纹

过扫描如何工作​

过扫描延伸刀具路径在每条光栅线之前和之后：

过程：

  1. **引入：**激光移动到线条开始_之前_的位置
  2. **加速：**激光加速到目标速度（激光关闭）
  3. **雕刻：**激光开启并以恒定速度雕刻
  4. **减速：**激光关闭并在线条结束_之后_减速

**结果：**整个雕刻区域在恒定速度下获得一致的功率。

好处：

  • 整条光栅线的雕刻深度均匀
  • 无浅色/深色边缘
  • 更高质量的照片雕刻
  • 专业外观的结果

配置过扫描​

过扫描是 Rayforge 工作流程管道中的变换器。

启用：

  1. 选择图层带有光栅雕刻
  2. 打开工作流程设置（或操作设置）
  3. 添加过扫描变换器如果尚未存在
  4. 配置距离

设置：

设置描述典型值
启用切换过扫描开/关开（用于光栅）
距离 (mm)延伸线条多远2-5 mm

选择过扫描距离​

过扫描距离应允许机器完全加速到目标速度。

实用指南：

最大速度加速度推荐过扫描
3000 mm/min (50 mm/s)低5 mm
3000 mm/min (50 mm/s)中3 mm
3000 mm/min (50 mm/s)高2 mm
6000 mm/min (100 mm/s)低10 mm
6000 mm/min (100 mm/s)中6 mm
6000 mm/min (100 mm/s)高4 mm

影响所需距离的因素：

  • **速度：**更高的速度 = 需要更多距离来加速
  • **加速度：**更低的加速度 = 需要更多距离
  • **机器机械：**皮带驱动与直接驱动影响加速度

调整：

  • **太少：**边缘仍可见加速标记
  • **太多：**浪费时间，可能超出机器边界
  • 从 3mm 开始并根据结果调整

测试过扫描设置​

测试过程：

  1. 创建测试雕刻：

    • 实心填充矩形（50mm x 20mm）
    • 使用您典型的雕刻设置
    • 启用过扫描 3mm

  2. 雕刻测试：

    • 运行作业
    • 让它完成

  3. 检查边缘：

    • 查看矩形的左右边缘
    • 检查边缘的暗度变化
    • 比较边缘暗度与中心暗度

  4. 调整：

    • **如果边缘更浅/更深：**增加过扫描
    • **如果边缘与中心匹配：**过扫描足够
    • **如果边缘完美：**尝试略微减少过扫描以节省时间

何时使用过扫描​

始终用于：

  • 照片雕刻（光栅）
  • 填充图案
  • 任何高细节光栅工作
  • 灰度图像雕刻
  • 文本雕刻（光栅模式）

可选用于：

  • 矢量切割（不需要）
  • 非常慢的雕刻（加速度不太明显）
  • 大的简单形状（边缘不太关键）

禁用用于：

  • 矢量操作
  • 非常小的工作区域（可能超出边界）
  • 当边缘质量不重要时
