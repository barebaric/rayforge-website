波前

波前自适应清除使用从口袋中心向外扩展的同心刀具路径填充闭合矢量形状， 就像池塘中的涟漪。扩展的环自动处理内部岛屿，产生平滑、连续的刀路， 没有光栅扫描的急剧方向反转。

概述​

与前后扫描平行线的传统光栅雕刻不同，波前生成从每个口袋中心辐射的 同心通道。这产生均匀、涟漪般的光洁度，非常适合填充图案本身对视觉 效果有贡献的应用。

波前操作：

  • 用同心通道填充闭合矢量形状（口袋）
  • 从口袋中心向外扩展
  • 自动绕过内部岛屿（口袋内的孔洞）
  • 产生无方向反转的平滑刀路

何时使用波前​

波前是口袋区域的替代填充图案。其同心环比平行光栅线在视觉上更 令人愉悦，扩展图案自然地与圆形或有机形状相得益彰。

使用波前自适应清除用于：

  • 填充矢量设计中的口袋
  • 印章和模具制作——波前清除背景口袋，同时将凸起特征保留为内部岛屿
  • 填充纹理在成品中可见的应用

不要使用波前用于：

  • 沿轮廓切割（改用轮廓）
  • 填充位图图像（改用雕刻）
  • 不存在口袋的薄壁区域

创建波前操作​

第1步：选择对象​

  1. 在画布上导入或绘制闭合矢量形状
  2. 选择定义口袋边界的对象
  3. 确保形状是闭合路径

第2步：添加波前操作​

  • 菜单： 操作 → 添加波前
  • 右键： 上下文菜单 → 添加操作 → 波前

第3步：配置设置​

调整步进和偏移以匹配您的材料和所需光洁度。

关键设置​

步进（Step Over）​

连续波前通道之间的距离（毫米）。较小的值提供更密集的覆盖，更多 通道和更长的作业时间。较大的值使通道间距更大，完成速度更快。

步进默认为激光光斑大小，范围为 0.05–50.0 毫米。

步进线密度作业时间
0.1 毫米密集，很多线最慢
0.3 毫米中等中等
1.0 毫米+稀疏，较少线快速

大多数应用的典型值为 0.1–0.5 毫米。

偏移（Offset）​

与口袋壁的额外间隙（毫米）。在最外层波前通道和边界轮廓之间创建 边距。这在单独的轮廓通道将完成边缘时很有用，或者当您 想在口袋周围留下刻意边框时。

范围：0.0–20.0 毫米。默认值为 0.0（波前通道延伸到边界）。

波前如何工作​

  1. 进入通道——螺旋进入切入口袋中心，建立初始清除区域
  2. 波前扩展——从清除的中心开始，同中心环向外扩展。每个环超出 前一个环配置的步进距离
  3. 岛屿处理——随着波前增长，它遇到并绕过任何内部岛屿，使其保留
  4. 完成——扩展继续，直到整个口袋区域被覆盖

后处理​

波前操作支持：

  • 路径平滑——减少刀路中的锯齿状边缘
  • 路径优化——最小化通道之间的移动距离

技巧与最佳实践​

选择步进​

  • 更密集的覆盖（小步进）意味着更多通道和更长的作业时间
  • 稀疏覆盖（大步进）更快，但在通道之间留下更多材料
  • 根据您的应用平衡密度和作业时间

印章和模具制作​

波前非常适合印章制作。扩展的同心环自然地清除背景口袋，同时绕过 被视为内部岛屿的凸起特征。

与轮廓结合​

常见的工作流程是用波前清除口袋内部，然后用轮廓通道完成 边界以获得干净的边缘。设置偏移为轮廓切割留出足够的边距。
