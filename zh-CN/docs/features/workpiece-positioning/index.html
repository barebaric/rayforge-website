工件定位指南

本指南涵盖了 Rayforge 中所有可用于精确定位工件和在切割或雕刻前对齐设计的方法。

概述​

精确的工件定位对于以下方面至关重要：

  • 防止浪费：避免在错误的位置切割
  • 精确对齐：在预打印材料上定位设计
  • 可重复的结果：一致地多次运行相同的作业
  • 多部件作业：在单张材料上对齐多个部件

Rayforge 提供了几种互补的定位工具：

方法用途最适用于
对焦模式查看激光位置快速视觉对齐
边框预览预览作业边界验证设计是否适合材料
WCS 零点设置坐标原点可重复定位
相机叠加视觉设计放置在现有特征上精确对齐

对焦模式（激光指示器）​

对焦模式以低功率级别开启激光，充当"激光指示器"，帮助您准确查看激光头所在的位置。

启用对焦模式​

  1. 连接到您的机器
  2. 点击工具栏中的对焦按钮（激光图标）
  3. 激光以配置的对焦功率级别开启
  4. 移动激光头以查看光束在材料上的位置
  5. 完成后再次点击对焦按钮以关闭
安全警告

即使在低功率下，激光也可能损伤眼睛。切勿直视光束或将其指向反射 表面。佩戴适当的护目装备。

配置对焦功率​

对焦功率决定了激光点的亮度：

  1. 转到 设置 → 机器 → 激光
  2. 找到 对焦功率 设置
  3. 设置一个使点可见但不会标记材料的值
    • 典型值：大多数材料为 1-5%
    • 设置为 0 以禁用该功能
寻找合适的功率

从 1% 开始并逐渐增加。点应该可见但不会在材料上留下任何痕迹。 较暗的材料可能需要更高的功率才能清楚地看到点。

何时使用对焦模式​

  • 快速对齐检查：查看激光是否大致在您期望的位置
  • 查找材料边缘：移动到角落以验证材料放置
  • 设置 WCS 原点：在设置 WCS 之前将激光定位到所需的零点
  • 验证原点位置：检查归位是否正常工作

边框预览​

边框预览以低（或零）功率追踪作业的边界矩形，准确显示设计将被 切割或雕刻的位置。

如何边框预览​

  1. 在 Rayforge 中加载并定位您的设计
  2. 点击 机器 → 边框 或按 Ctrl+F
  3. 激光头追踪作业的边界框
  4. 验证轮廓是否适合您的材料

边框设置​

在 设置 → 机器 → 激光 中配置边框行为：

  • 边框速度：边框期间激光头移动的速度（较慢 = 更容易看到）
  • 边框功率：边框期间的激光功率
    • 设置为 0 进行空跑边框（激光关闭，仅移动）
    • 设置为 1-5% 在材料上留下可见痕迹
空跑边框与低功率边框
  • 空跑边框（0% 功率）：对任何材料都安全，但您只能看到头部 移动
  • 低功率边框：留下微弱的可见痕迹，适用于在深色材料上精确 对齐

何时边框预览​

  • 每次作业前：快速验证设计是否适合
  • 位置更改后：确认新位置正确
  • 昂贵材料：在执行前再次检查
  • 多部件作业：验证所有部件都适合材料

有关更多详细信息，请参阅 作业边框预览。

设置 WCS 零点（工件坐标系）​

工件坐标系（WCS）允许您为作业定义自定义"零点"。这使得将作业 对齐到材料位置变得容易。

快速 WCS 设置​

  1. 移动激光头到材料的角落（或所需的原点）
  2. 打开控制面板（Ctrl+L）
  3. 选择一个 WCS（G54 是默认的工件坐标系）
  4. 点击 零点 X 和 零点 Y 将当前位置设置为原点
  5. 您的设计的 (0,0) 点现在将与该位置对齐

理解坐标系​

Rayforge 使用多个坐标系：

系统描述
G53机器坐标（固定，无法更改）
G54工件坐标系 1（默认）
G55-G59附加工件坐标系
多个工作区域

为不同的夹具位置使用不同的 WCS 插槽。例如：

  • G54 用于床的左侧
  • G55 用于右侧
  • G56 用于旋转附件

何时设置 WCS 零点​

  • 新材料放置：将原点对齐到材料角落
  • 夹具工作：将原点设置为夹具参考点
  • 可重复作业：相同的作业，不同的位置
  • 生产运行：多个工件的一致定位

有关完整文档，请参阅 工件坐标系。

基于相机的定位​

相机叠加显示材料的实时视图，您的设计叠加在上面，实现精确的视觉 对齐。

设置相机​

  1. 在工作区域上方连接 USB 相机
  2. 转到 设置 → 相机 并添加您的相机设备
  3. 启用相机以在画布上查看叠加
  4. 对齐相机使用对齐程序（准确定位所需）

相机对齐​

相机对齐将相机像素映射到现实世界坐标：

  1. 打开 相机 → 对齐相机
  2. 在已知位置放置对齐标记（至少 4 个点）
  3. 输入每个点的现实世界 X/Y 坐标
  4. 点击 应用 计算变换
对齐精度
  • 使用分布在整个工作区域的点
  • 用尺子仔细测量世界坐标
  • 使用机器位置（移动到已知坐标）以获得最佳精度

使用相机叠加定位​

  1. 启用相机叠加以查看您的材料
  2. 导入您的设计
  3. 拖动设计以与相机中可见的特征对齐
  4. 微调使用箭头键进行像素级精确定位
  5. 边框预览以验证在运行作业之前

何时使用相机定位​

  • 预打印材料：将切割对齐到现有打印
  • 不规则材料：在非矩形件上定位
  • 精确放置：亚毫米精度要求
  • 复杂布局：具有特定间距的多个元素

有关完整文档，请参阅 相机集成。

推荐工作流程​

基本定位工作流程​

对于矩形材料上的简单作业：

  1. 放置材料在激光床上
  2. 启用对焦模式并移动以验证材料位置
  3. 在材料角落设置 WCS 零点
  4. 在画布中定位您的设计
  5. 边框预览作业以验证放置
  6. 运行作业

精确对齐工作流程​

对于在预打印或标记材料上的精确放置：

  1. 设置并对齐相机（一次性设置）
  2. 放置材料在激光床上
  3. 启用相机叠加以查看材料
  4. 在相机图像上视觉导入并定位设计
  5. 禁用相机并边框预览以验证
  6. 运行作业

生产工作流程​

用于运行多个相同的作业：

  1. 在激光床上设置夹具
  2. 设置与夹具对齐的 WCS 零点（例如 G54）
  3. 加载并配置您的设计
  4. 边框预览以验证与夹具的对齐
  5. 运行作业
  6. 更换材料并重复（WCS 保持不变）

多位置工作流程​

用于在不同位置运行相同的作业：

  1. 设置多个 WCS 位置：
    • 移动到位置 1，设置 G54 零点
    • 移动到位置 2，设置 G55 零点
    • 移动到位置 3，设置 G56 零点
  2. 加载您的设计（所有位置使用相同的设计）
  3. 选择 G54，边框预览并运行
  4. 选择 G55，边框预览并运行
  5. 选择 G56，边框预览并运行

故障排除​

对焦模式下激光点不可见​

  • 在激光设置中增加对焦功率
  • 深色材料可能需要更高的功率（5-10%）
  • 检查激光连接并确保机器有响应
  • 验证对焦功率未设置为 0

相机叠加未对齐​

  • 使用更多参考点重新运行相机对齐
  • 检查相机安装 - 它可能已移动
  • 验证世界坐标是否准确测量
  • 参阅相机故障排除在相机集成文档中
