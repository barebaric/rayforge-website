画布工具

画布提供一套全面的工具，用于操作设计、测量和准备激光作业。

选择工具​

选择、移动和变换画布上的对象。

用法：

  • 点击选择单个对象
  • ctrl + 点击选择多个对象
  • 拖动创建选择框
  • 点击并拖动选定的对象以移动

拖动时的修饰键：

在拖动时按住修饰键会改变行为：

  • ctrl 在移动时：将对象对齐到5mm网格，便于精确定位
  • ctrl 在旋转时：将旋转对齐到5度增量，保持角度一致
  • shift 在缩放时：约束比例（均匀缩放）
  • ctrl 在缩放时：从中心缩放而不是从相对边缘缩放

变换手柄：

  • 角手柄：按比例缩放（按住 shift 以非比例缩放）
  • 边手柄：在一个方向缩放
  • 旋转手柄：在选定的对象上点击一次以切换旋转模式，然后拖动以围绕中心点旋转

键盘快捷键：

  • arrow-keys：将选定的对象移动 1 单位
  • shift+arrow-keys：移动 10 单位
  • ctrl+d：复制选择

平移工具​

在画布上导航而不会意外移动对象。

用法：

  • 点击并拖动以平移
  • 或者，按住 space 并拖动，任何工具活动时都可以

缩放工具​

放大设计特定区域。

用法：

  • 点击以在点处放大
  • alt + 点击以缩小
  • 点击并拖动以缩放到特定区域

快捷键：

  • ctrl+"+"：放大
  • ctrl+"-"：缩小
  • ctrl+0：重置缩放（适应窗口）
  • 鼠标滚轮：在光标位置放大/缩小

测量工具​

在画布上测量距离和角度。

用法：

  • 点击起点
  • 点击终点完成测量
  • 以当前单位（mm 或英寸）显示测量值

功能：

  • 两点之间的距离
  • 相对于水平的角度
  • 测量时实时预览

对齐工具​

对齐和分布多个对象。

对齐选项：

  • 左边缘对齐
  • 右边缘对齐
  • 上边缘对齐
  • 下边缘对齐
  • 水平居中对齐
  • 垂直居中对齐

分布选项：

  • 水平分布
  • 垂直分布
  • 等间距

用法：

  1. 选择多个对象
  2. 从工具栏或编辑菜单选择对齐/分布选项
  3. 对象立即对齐/分布

网格和捕捉​

使用网格和捕捉帮助精确定位。

网格：

  • 切换：视图 → 显示网格（ctrl+g）
  • 在首选项中配置间距
  • 仅视觉参考（可选捕捉）

捕捉：

  • 捕捉到网格：将对象对齐到网格点
  • 捕捉到对象：对齐到其他对象的边缘
  • 捕捉到原点：对齐到机器原点 (0,0)

切换捕捉：视图 → 捕捉（ctrl+shift+g）

对象变换​

以数字方式变换对象以获得精度。

通过属性面板访问：

  • 位置 (X, Y)：精确坐标
  • 大小 (W, H)：精确尺寸
  • 旋转：距水平的度数
  • 缩放：原始大小的百分比

布尔操作​

组合或减去形状：

  • 并集：合并重叠的形状
  • 差集：从另一个形状减去一个形状
  • 交集：仅保留重叠区域
  • 排除：删除重叠区域

用法：

  1. 选择两个或更多矢量对象
  2. 从编辑菜单选择布尔操作
  3. 结果替换选定的对象

高效使用画布的提示​

  1. 使用键盘快捷键：比工具栏点击快得多
  2. 掌握平移和缩放：对于大型或详细的设计至关重要
  3. 捕捉到网格：加速矩形布局的对齐
  4. 先测量：在生成 G 代码之前验证尺寸
  5. 组合相关对象：更容易移动和组织（ctrl+g 组合）
